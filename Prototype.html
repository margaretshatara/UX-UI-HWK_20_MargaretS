Prototype and Re-iteration

We used Figma, as a team, to work on the different prototypes and design. The project load was split up accordingly and we worked on the different layouts for both desktop and mobile applications. My main responsbility was putting together and prototyping the desktop interface along with creating the design guide for team members to follow.

Main User Interface should :

  • Incorporate the colours that work as the Canadian Cancer Society Brand
  • Not lose Canadian Cancer Societies Vision
  • Provide a 'positive' looking interface to make user experience better
  • Be easy to navigate through and use
